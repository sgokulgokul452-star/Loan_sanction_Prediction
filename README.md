# Loan Sanction Prediction

This project predicts whether a loan will be approved using Machine Learning.

## Project Overview
The model analyzes applicant information such as income, credit history, and loan amount to predict loan approval.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Machine Learning Model
Random Forest Classifier

## Steps in the Project
1. Data preprocessing
2. Handling missing values
3. Feature encoding
4. Model training
5. Model evaluation
